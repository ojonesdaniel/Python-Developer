# Funções em Python

As funções são blocos de código reutilizáveis que executam uma tarefa específica.  
Elas ajudam a organizar o código, evitar repetição e melhorar a legibilidade.

---

# 1️⃣ O que são Funções?

Uma função é um conjunto de instruções que pode ser executado sempre que for chamado.

```python
def saudacao():
    print("Olá, mundo!")
```

Para executar a função:

```python
saudacao()
```

---

# 2️⃣ Estrutura de uma Função

```python
def nome_da_funcao(parametros):
    # bloco de código
    return valor
```

- `def` → palavra-chave para definir função
- `nome_da_funcao` → nome da função
- `parametros` → valores de entrada (opcional)
- `return` → valor de saída (opcional)

---

# 3️⃣ Função sem Parâmetros

```python
def mensagem():
    print("Bem-vindo ao sistema!")

mensagem()
```

---

# 4️⃣ Função com Parâmetros

Parâmetros permitem passar valores para a função.

```python
def saudacao(nome):
    print("Olá,", nome)

saudacao("Daniel")
```

---

# 5️⃣ Função com Retorno (`return`)

A função pode retornar um valor.

```python
def soma(a, b):
    return a + b

resultado = soma(10, 5)
print(resultado)
```

---

# 6️⃣ Múltiplos Parâmetros

```python
def dados(nome, idade):
    print("Nome:", nome)
    print("Idade:", idade)

dados("Daniel", 20)
```

---

# 7️⃣ Parâmetros com Valor Padrão

```python
def saudacao(nome="Usuário"):
    print("Olá,", nome)

saudacao()
saudacao("Daniel")
```

---

# 8️⃣ Retornando Múltiplos Valores

```python
def operacoes(a, b):
    return a + b, a - b

soma, sub = operacoes(10, 5)
print(soma, sub)
```

---

# 9️⃣ Escopo de Variáveis

Variáveis criadas dentro da função só existem dentro dela.

```python
def teste():
    x = 10
    print(x)

teste()
# print(x) -> erro
```

---

# 🔟 Funções Built-in (Funções Prontas)

Python já possui várias funções prontas:

- `print()` → saída de dados
- `input()` → entrada de dados
- `len()` → tamanho
- `type()` → tipo da variável
- `int()`, `float()`, `str()` → conversões

```python
print(len("Python"))
```

---

# 1️⃣1️⃣ Boas Práticas

- Use nomes descritivos
- Evite funções muito grandes
- Uma função deve ter apenas uma responsabilidade
- Utilize `return` ao invés de apenas `print` quando possível
- Evite repetir código

---

# 1️⃣2️⃣ Exemplo Completo

```python
def calcular_media(n1, n2):
    media = (n1 + n2) / 2
    return media

nota1 = float(input("Digite a primeira nota: "))
nota2 = float(input("Digite a segunda nota: "))

resultado = calcular_media(nota1, nota2)

print("Média:", resultado)
```
