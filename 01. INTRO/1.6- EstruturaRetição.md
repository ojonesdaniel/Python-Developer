# Estruturas de Repetição em Python

As estruturas de repetição permitem executar um bloco de código múltiplas vezes, de forma controlada. São fundamentais para percorrer coleções, processar dados em lote e automatizar tarefas repetitivas.

---

## 1. Estrutura `for`

O `for` é utilizado quando sabemos previamente o número de repetições ou quando queremos percorrer elementos de uma sequência (lista, tupla, string, `range`, etc.).

```python
for variavel in sequencia:
    # bloco de código executado a cada iteração
```

### Exemplo com `range`

```python
for numero in range(1, 6):
    print(numero)
```

Saída esperada:

```
1
2
3
4
5
```

O `range(1, 6)` gera números de 1 até 5 (o valor final não é incluído).

---

## 2. Função `range()`

A função `range()` é frequentemente utilizada com o `for`.

### Sintaxes possíveis

```python
range(fim)
range(inicio, fim)
range(inicio, fim, passo)
```

### Exemplos

```python
# De 0 até 4
for i in range(5):
    print(i)

# De 2 até 9
for i in range(2, 10):
    print(i)

# De 0 até 10 pulando de 2 em 2
for i in range(0, 11, 2):
    print(i)
```

---

## 3. Estrutura `while`

O `while` executa um bloco de código enquanto uma condição for verdadeira (`True`). É indicado quando não sabemos exatamente quantas repetições serão necessárias.

```python
while condição:
    # bloco executado enquanto condição for True
```

### Exemplo

```python
contador = 1

while contador <= 5:
    print(contador)
    contador += 1
```

⚠ Atenção: se a condição nunca se tornar falsa, ocorre um **loop infinito**.

---

## 4. Controle de Loop: `break`

O `break` interrompe imediatamente a execução do loop.

```python
for numero in range(1, 11):
    if numero == 5:
        break
    print(numero)
```

Saída:

```
1
2
3
4
```

---

## 5. Controle de Loop: `continue`

O `continue` pula a iteração atual e passa para a próxima.

```python
for numero in range(1, 6):
    if numero == 3:
        continue
    print(numero)
```

Saída:

```
1
2
4
5
```

---

## 6. Estrutura `else` em Loops

Tanto `for` quanto `while` podem ter um bloco `else`, executado quando o loop termina normalmente (sem `break`).

```python
for numero in range(3):
    print(numero)
else:
    print("Loop finalizado com sucesso.")
```

Se houver `break`, o `else` não será executado.

---

## 7. Loops Aninhados

É possível colocar um loop dentro de outro.

### Exemplo

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"i={i}, j={j}")
```

Esse padrão é comum em matrizes, tabelas e algoritmos de comparação.

---

## 8. Percorrendo Listas

```python
nomes = ["Ana", "Carlos", "Maria"]

for nome in nomes:
    print(nome)
```

Com índice usando `enumerate`:

```python
for indice, nome in enumerate(nomes):
    print(indice, nome)
```

---

## 9. Boas Práticas

- Evite loops infinitos garantindo que a condição do `while` seja alterada.
- Prefira `for` quando souber o número de iterações.
- Use nomes de variáveis significativos.
- Evite aninhamentos excessivos.
- Utilize `break` e `continue` com critério para manter legibilidade.

---

## 10. Exemplo Completo

```python
numeros = []

for i in range(5):
    valor = int(input("Digite um número: "))
    numeros.append(valor)

soma = 0
contador = 0

while contador < len(numeros):
    soma += numeros[contador]
    contador += 1

media = soma / len(numeros)

print(f"Lista digitada: {numeros}")
print(f"Soma: {soma}")
print(f"Média: {media}")
```

---

## 11. Comparação entre `for` e `while`

| Estrutura | Quando usar |
|------------|------------|
| `for`      | Quando há sequência definida ou número conhecido de repetições |
| `while`    | Quando a repetição depende de uma condição dinâmica |

---

## 12. Erros Comuns

- Esquecer de atualizar a variável no `while`.
- Usar `range` com limite incorreto.
- Criar loops infinitos sem intenção.
- Alterar a lista enquanto a percorre sem controle.

---

As estruturas de repetição são fundamentais para qualquer aplicação em Python. Dominá-las permite escrever algoritmos mais eficientes, organizados e reutilizáveis.
